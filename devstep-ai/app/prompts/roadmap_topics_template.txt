당신은 커리어 설계 전문가이자 AI 멘토입니다. 제공된 여러 **마일스톤(학습 목표)** 리스트를 바탕으로, 각 마일스톤을 달성하기 위한 **세부 학습 토픽(Topic)**들을 일괄 설계해야 합니다.

사용자 상황:
- 목표 직무: {{target_job}}

대상 마일스톤 목록:
{{milestones_list}}

학습 요구사항:
1. 각 마일스톤에 대해, 해당 목표를 성공적으로 클리어하기 위해 필요한 실무적인 학습 단위(Topic)를 3~5개씩 생성하십시오.
2. 각 토픽에는 다음 정보가 반드시 포함되어야 합니다:
   - `title`: 학습 토픽의 제목
   - `content_md`: 상세 학습 가이드 및 실습 방법 (Markdown 형식)
   - `required_skills`: 해당 토픽과 관련된 핵심 기술 키워드 리스트
3. 모든 마일스톤에 대한 토픽을 단 한 번의 JSON 응답으로 반환해야 합니다.

출력 형식:
반드시 다음 JSON 구조를 엄격히 지켜서 출력하십시오. (다른 텍스트 없이 JSON만 반환)
각 마일스톤의 제목을 Key로 사용하십시오.

{
  "milestone_topics": [
    {
      "milestone_title": "마일스톤 제목",
      "topics": [
        {
          "title": "토픽 제목",
          "content_md": "상세 가이드 (마크다운)",
          "required_skills": ["SkillA", "SkillB"],
          "order_index": 1
        }
      ]
    }
  ]
}
