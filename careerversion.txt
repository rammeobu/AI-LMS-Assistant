상세 기획서 AI 기반 커리어 포트폴리오 분석 및 추천 플랫폼: AI Career Navigator
1. 프로젝트 개요 및 배경
1.1 기획 목적 및 배경
현재 대학생들은 인턴십, 대외활동, 자격증, 공모전 등 무수히 많은 커리어 요소들을 준비해야 하는 환경에 놓여 있습니다. 그러나 정보의 과잉 속에서 정작 자신에게 어떤 역량이 부족한지 객관적으로 파악하지 못하거나, 수많은 활동 중 무엇을 먼저 해야 할지 우선순위를 설정하는 데 큰 어려움을 겪고 있습니다. 본 프로젝트는 이러한 문제를 해결하기 위해 AI가 사용자의 포트폴리오를 정밀 분석하여 역량 격차를 진단하고, 개인의 상황에 최적화된 학습 로드맵과 활동을 추천함으로써 체계적이고 효율적인 커리어 성장을 지원하고자 합니다.
1.2 서비스 한 줄 소개
“AI가 포트폴리오를 분석해 부족한 역량을 진단하고, 대외활동·자격증·공모전·학습 계획까지 추천해주는 대학생 맞춤형 커리어 성장 플랫폼”
1.3 프로젝트 목표
핵심 목표:  포트폴리오 AI 분석을 통한 역량 상태 파악 및 맞춤형 커리어 보완 활동 추천 서비스 제공
세부 목표:
포트폴리오 기반 AI 정밀 역량 분석 서비스 제공
사용자 직무 목표에 따른 학습 로드맵 자동 생성 및 추천
개인별 맞춤 자격증, 대외활동, 공모전 큐레이션
활동 및 학습 계획의 체계적 관리를 위한 일정 관리 기능
동일 목표를 가진 사용자 간의 팀 빌딩 및 네트워킹 지원
시각화된 대시보드를 통한 직관적인 커리어 성장 현황 제공
2. 타겟 사용자 및 핵심 컨셉
2.1 타겟 사용자 분석
취업 준비 중인 대학생:  현재 자신의 역량을 객관적으로 진단하고 실질적인 취업 전략을 수립하고자 하는 사용자
활동 방향 설정이 필요한 학생:  실무 역량 강화에 도움이 되는 자격증이나 활동이 무엇인지 선별하는 데 어려움을 느끼는 사용자
포트폴리오 체계화 희망자:  분산된 활동 이력을 통합 관리하고 AI를 통해 전문적인 피드백을 얻고자 하는 사용자
효율적 관리 희망자:  학습 일정과 커리어 준비를 단일 플랫폼에서 관리하여 생산성을 극대화하고 싶은 사용자
2.2 서비스 핵심 컨셉: "내 상황에 맞는 커리어 성장 경로 제시"
AI는 사용자의 입력 데이터를 기반으로 단순 정보 제공을 넘어 다음의 7가지 핵심 인사이트를 도출합니다.
포트폴리오 내 강점 및 약점의 정밀 분석 결과
목표 직무 역량 대비 현재 보유 역량의 격차(Gap) 진단
역량 보완을 위한 맞춤형 자격증 추천
직무 연관도가 높은 대외활동 및 공모전 추천
개인별 맞춤형 단계별 학습 로드맵
실무 대비를 위한 인턴십 준비 가이드
실행력 제고를 위한 일정 관리 및 활동 계획 기능

3.2 랜딩 및 메인 대시보드
대기화면(Landing Page):  서비스 목적 및 AI 기능 소개, 추천 서비스 미리보기, 유입 유도 버튼, 실시간 모집 중인 활동 카드, 성공 사례 전시.
프로필(메인화면):  사용자의 커리어 상태를 통합 제공하는 허브 페이지.
사용자 기본 프로필 정보
관심 분야 및 목표 직무 정보
현재 보유 기술 스택(Tech Stack)
자격증 보유 현황 및 상태
프로젝트 및 대외활동 요약 리스트
AI 분석 결과 요약(강점/약점)
추천 활동 요약 카드
캘린더 위젯:  AI가 추천한 일정 및 마감 임박 공고가 자동 동기화되어 표시됨.
3.3 AI 포트폴리오 분석 및 입력
포트폴리오 입력 항목:  학력, 프로젝트 경험, 대외활동, 공모전, 자격증, 기술 스택, 수상 경력, 자기소개 요약 등.
AI 6대 평가 항목:
목표 직무 대비 기술 역량 부족 여부
실무 경험의 충분성 분석
직무 관련 필수 자격증 보완 필요성
프로젝트의 다양성 및 전문성
대외활동 경험의 적절성
일정 관리 부족 여부 및 계획성 평가
분석 결과 예시:
강점:  "프로젝트 경험 3건 보유, 실무 협업 및 Git 활용 능력 우수"
약점:  "목표 직무 관련 국가공인 자격증 부재, IT 분야 대외활동 경험 부족"
3.4 맞춤형 추천 서비스
5대 추천 대상:  자격증, 대외활동, 공모전, 학습 콘텐츠(강의/주제), 인턴십 준비 항목.
추천 방식 로직:
사용자 설정 목표 직무 기반의 역량 매칭
현재 포트폴리오 분석 결과 도출된 '부족 요소' 기반 필터링
사용자 행동 데이터 기반의 관심 분야 추천
3.5 학습 및 실행 지원 (Backend Developer 예시)
학습 로드맵:  목표 직무에 따른 단계별(초급/중급/심화) 가이드 제공.
예시:  Java 기초 → 객체지향 프로그래밍 → Spring Boot → DB 설계 → REST API → 배포 경험 → 포트폴리오 완성
인턴십 가이드:  직무별 이력서/자소서 팁, 면접 체크리스트 및 필요 기술 스택 안내.
예시:  Java/Spring/DB 역량 보완 필요, GitHub 프로젝트 관리 강화, SQLD 자격증 추천 등.
캘린더 기능:  주/월간 보기, 공모전/자격증 마감일 자동 표시, AI 추천 일정 반영(예: "이번 주 SQLD 공부 3회"), 개인 일정 관리.
3.6 커뮤니티 및 정보 제공
팀원 모집 리스트:  같은 활동에 관심 있는 사용자 목록 제공. 모집 상태, 목표 직무, 관심 분야, 간단한 자기소개 및 팀원 모집 여부 확인 가능.
공모전 정보 분류 체계:  '현재 모집 중', '모집 예정', '마감 임박', '관심 등록 공고'로 분류.
상세 데이터 필드:  제목, 모집 기간, 분야,  참가 대상 ,  관련 역량 ,  AI 추천 이유 .

5. 기술 스택 및 시스템 아키텍처
5.1 프론트엔드 및 백엔드 설계
Frontend:   Next.js , TypeScript, Tailwind CSS 기반의 반응형 UI.  Zustand 를 활용한 가벼운 상태 관리 및  shadcn/ui 를 통한 일관된 컴포넌트 설계.
Backend:
Next.js API Routes:  가벼운 API 처리 및 BFF(Backend For Frontend) 역할 수행.
Spring Boot:  데이터 처리의 핵심 로직, 복잡한 비즈니스 규칙 및 AI 분석 요청 처리 전담.
Database:   PostgreSQL  또는 MySQL (사용자, 포트폴리오, 추천 결과, 일정 데이터 저장).
5.2 데이터베이스 설계 (Draft)
-- 사용자 정보 테이블
Table User {
    user_id UUID [primary key]
    name VARCHAR
    email VARCHAR
    password VARCHAR
    school VARCHAR
    major VARCHAR
    grade INT
    target_job VARCHAR
}
-- 포트폴리오 데이터 테이블
Table Portfolio {
    portfolio_id UUID [primary key]
    user_id UUID [ref: > User.user_id]
    skills TEXT
    certifications TEXT
    projects TEXT
    activities TEXT
    awards TEXT
    intro TEXT
}
-- AI 분석 결과 테이블
Table AnalysisResult {
    analysis_id UUID [primary key]
    user_id UUID [ref: > User.user_id]
    strengths TEXT
    weaknesses TEXT
    missing_skills TEXT -- '부족역량' 필드
    recommendation TEXT -- '추천내용' 필드
    created_at TIMESTAMP -- '생성일' 필드
}
-- 공모전 및 대외활동 정보
Table Activity {
    activity_id UUID [primary key]
    title VARCHAR
    category VARCHAR
    recruit_period VARCHAR
    description TEXT
    required_skills TEXT -- '필요역량' 필드
}
5.3 AI 기술 구현 전략
포트폴리오 분석:  OpenAI API를 활용하여 입력된 텍스트 데이터를 분석하고 직무별 표준 요구 역량과 매칭.
추천 엔진 (Hybrid Approach):
Rule-based:  자격증 취득 요건, 필수 기술 스택 등 명확한 조건 기반 추천.
Vector Search:  사용자의 포트폴리오 의미(Semantic)를 분석하여 유사한 합격 사례나 가장 적합한 직무/공모전을 매칭하는 시맨틱 검색 활용.
6. 차별점 및 기대효과
6.1 서비스 차별화 전략
역량 격차 분석 중심:  단순 공고 나열이 아닌, AI를 통해 현재 내 상태와 목표 간의 격차를 진단하는 데 집중함.
맞춤형 연계성:  분석 결과가 단순 리포트에서 끝나지 않고, 즉각적인 학습 로드맵 및 활동 추천으로 이어짐.
통합 관리 에코시스템:  학습 로드맵과 캘린더 일정을 연계하여 실행 가능성을 극대화함.
팀 빌딩 지원:  동일한 활동에 관심 있는 사용자 리스트를 제공하여 정보 소비를 넘어 실질적인 협업 기회를 창출함.
All-in-one 플랫폼:  포트폴리오 관리, 역량 분석, 커리어 추천, 협업 매칭을 하나의 플랫폼에서 해결함.
6.2 기대 효과
개인 측면:  진로 준비의 방향성 명확화, 부족 역량 조기 파악을 통한 시행착오 감소, 맞춤형 정보 필터링을 통한 시간 절약 및 취업 경쟁력 제고.
생태계 측면:  대학생들의 대외활동 및 공모전 참여 활성화, 사용자 간 협업 네트워크 확장을 통한 실무 중심의 커리어 준비 문화 확산.
7. 프로젝트 로드맵 (Development Schedule)
단계,주요 태스크,상세 내용
1단계: 기획 및 설계,요구사항 분석 및 설계,"기능 정의서 확정, UI/UX 화면 설계, DB 스키마 상세 설계"
2단계: 프론트엔드 개발,UI 및 클라이언트 로직 구현,"로그인/회원가입, 메인화면, 캘린더, 대시보드, 입력 폼 개발"
3단계: 백엔드 개발,서버 인프라 및 API 구축,"사용자 관리, 포트폴리오 저장, 일정 관리, 공모전 조회 API"
4단계: AI 기능 개발,분석 및 추천 로직 구현,"포트폴리오 분석 알고리즘, 추천 엔진 고도화, 결과 시각화 모듈"
5단계: 테스트 및 배포,안정화 및 런칭,"기능/통합 테스트, UI 개선, 클라우드 배포 및 유지보수"
